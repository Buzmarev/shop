Простой интернет магазин.

Написан на PHP с использованием архитектуры MVC. Внешний вид взят с готового шаблона.

Для запуска необходимо:

- импортировать себе базу данных "phpshop.sql";

- В файле components => DB.php нужно поенять настройки под свой хость и sql;

- на сервере должен быть установлен Rewrite-module.

На сайте реализовано:

- постраничный переход по списку товаров;

- переход по категориям товаровж

- корзина;

- личный кабинет, авторизация и вход на сайт;

- панель администратора (переход по относительной ссылке '/admin/'), управление товарами, категориями и заказами.
